# WAN_SIM

WAN Simulator para Linux y Raspberry.

## DashboardAPI-EC

La primera fase de la plataforma EdgeConnect Dashboard vive en
[`DashboardAPI-EC/`](DashboardAPI-EC/README.md). Incluye el esqueleto
instalable de backend FastAPI, frontend React, compatibility layer,
PostgreSQL/TimescaleDB, Redis, Celery y Nginx.
